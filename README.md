# Bharat Properties CRM

A professional, multichannel CRM dashboard and campaign wizard designed for real estate management.

## Features

*   **Multichannel Campaign Wizard:**
    *   Unified creation flow for SMS, Email, WhatsApp, and RCS campaigns.
    *   **Creative Tools:** Built-in emoji picker, variable personalization (`{Name}`), and drag-and-drop media zones.
    *   **Real-time Previews:** High-fidelity sidebar mockups for mobile and desktop views.
    *   **Foolproof Layout:** Robust flexbox design ensuring navigation buttons are always accessible.

*   **Contact Management:**
    *   Detailed list views with avatar support.
    *   Smart filtering and group management.

## Tech Stack
*   **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (Vanilla)
*   **Design:** Custom "Bharat Properties" Design System (Clean, Professional, White-Label ready)

## How to Run
Simply open `index.html` in any modern web browser.
